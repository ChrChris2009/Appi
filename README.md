## 🚀 Guide Rapide de Déploiement

### Option A : Déploiement sur Vercel (Recommandé - Ultra rapide & gratuit)
1. Installez l'outil en ligne de commande : `npm i -g vercel`
2. Connectez-vous : `vercel login`
3. Lancez le déploiement en écrivant simplement `vercel` à la racine de votre projet.

### Option B : Déploiement sur Render (Idéal pour les serveurs Express persistants)
1. Créez un compte sur **Render.com** et liez votre compte GitHub.
2. Cliquez sur **New +** puis sélectionnez **Web Service**.
3. Liez votre dépôt Git et configurez ces deux options :
   - **Build Command :** `npm install`
   - **Start Command :** `node index.js`
